# CPU Project

## Numbers
- Int - Double
#### Initialization
- Number x;
#### Assignment
- x = 5; or x = 5.5;
#### Capability
- EQUAL, AND, OR, NOT Operations 
- +, -, /, * Arithmetic Operations

## Word
- Char - String
#### Initialization
- Word x;
#### Assignment
- x = "word"; or x = 'w';
#### Capability
- EQUAL, AND, OR, NOT Operations 
- Addition Arithmetic Operation (Concatenation)

## Output
- output << n 
- Words with both Number and Word
